Getting Started

Hello! We're Michael Stiffler and Daniel Wills! This is our combined CIS 408 and CIS 492 project. For this project, we used NASA's data set for landslides (which can be found here: Here) to predict whether or not a landslide at a user-selected area would be of risk. We accomplish this using a machine learning algorithm called k-nearest neighbors and our algorithm classifies a location as being either low-risk, medium-risk, or high-risk. Some of the tools we used to complete this are: Javascript, Python, HTML, P5, AJAX, Flask, SQL, Leaflet, Scikit-learn, Waitress, and much more!

Tutorial

When you're under the "Predict" tab and you would like to run the model, you can hold down CONTROL followed by a LEFT-CLICK to select a location. After a few seconds, the model will return the prediction for the selected location with a marker.

Predict

The "Predict" tab is used to run the k-nearest neighbors algorithm via a user-selected location (see Tutorial). Along with this, the landslides from the data set will be shown will their related information.

Sandbox

The "Sandbox" tab is a model-free space where the user can explore the map however they would like. This mode enables dragging so it's easier to navigate the map and explore each of the locations where there is a landslide.
